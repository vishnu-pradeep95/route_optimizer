1. **Add the helper function** near the top of main.py (after the imports, before middleware -- around line 65-80 area, near other utility code). Create:

```python
def _point_coords(geom: object) -> tuple[float, float] | None:
    """Extract (latitude, longitude) from a PostGIS geometry column.

    Centralizes the WKBElement → shapely Point → (lat, lng) conversion.
    Handles None geometry columns and provides proper type narrowing
    so callers don't need ``# type: ignore`` suppressions.

    Args:
        geom: A geoalchemy2 WKBElement (Point geometry) or None.

    Returns:
        (latitude, longitude) tuple, or None if geometry is None.
    """
    if geom is None:
        return None
    point = to_shape(geom)
    return (point.y, point.x)
```

Using `object` as the parameter type (the most permissive type) avoids importing WKBElement while still being type-safe. The `to_shape()` function accepts any WKBElement at runtime.

def _point_lat(geom: object) -> float | None:

    return float(to_shape(geom).x)
