Add visual indicators to the Driver PWA so drivers can see at a glance which delivery stops have approximate locations: a DaisyUI warning badge on the hero card, an orange notification dot on compact cards, and orange map pins for approximate stops.

Purpose: Drivers need situational awareness about navigation imprecision -- when the pin is approximate, they know to call the customer for directions rather than trusting the map blindly.

Output: Modified index.html with badge/dot rendering logic, rebuilt tailwind.css with DaisyUI badge styles, and orange map pin coloring for approximate stops.
