    **Section A: Extract area_name mapping (after preprocessing, before geocoding ~line 940)**

    After `orders = import_result.orders` and before the geocoding section, add:

    ```python
    # Phase 13: Build order_id -> area_name mapping for geocode validation.
    # CDCMS exports have an area_name column; standard CSVs do not.
    # This mapping is passed to the validator during geocoding so out-of-zone
    # addresses can be retried with the area name.
    area_name_map: dict[str, str] = {}
    if is_cdcms and not preprocessed_df.empty:
        area_name_map = dict(
            zip(
                preprocessed_df["order_id"].astype(str).str.strip(),
                preprocessed_df["area_name"].str.strip(),
            )
        )
    ```

    IMPORTANT: `preprocessed_df` is only defined when `is_cdcms` is True (line 918). For standard CSV uploads, `area_name_map` stays empty dict. The variable `preprocessed_df` is used after the `if is_cdcms:` block, so guard the access with the `is_cdcms` check.
