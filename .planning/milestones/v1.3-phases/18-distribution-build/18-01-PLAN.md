  <name>Task 1: Create build-dist.sh and update licensing docstring</name>
  <files>scripts/build-dist.sh, core/licensing/__init__.py</files>
  <action>
**Create `scripts/build-dist.sh`** following the architecture from 18-RESEARCH.md:

1. **Shebang and safety:** `#!/usr/bin/env bash` + `set -euo pipefail`

2. **Header comment block:** Match install.sh style. Describe what the script does, usage (`./scripts/build-dist.sh v1.3`), and output location.

3. **Version argument parsing:** Require exactly one argument. Print usage on missing arg:
   ```bash
   VERSION="${1:?Usage: $0 <version>  (e.g., v1.3)}"
   ```

4. **Color helpers:** Copy EXACTLY from install.sh lines 38-63 (the `if [ -t 1 ]` block and 5 helper functions with Unicode symbols). Do NOT retype -- match character-for-character.

5. **Staging directory with trap cleanup:**
   ```bash
   BUILD_DIR=$(mktemp -d)
   trap 'rm -rf "$BUILD_DIR"' EXIT
   DIST_NAME="kerala-delivery"
   STAGE="$BUILD_DIR/$DIST_NAME"
   ```

6. **rsync project tree with exclusions.** Use the COMPLETE exclusion list from RESEARCH.md section "Exclusion List (Complete)". IMPORTANT additions beyond RESEARCH.md:
   - Do NOT exclude `alembic.ini` -- Dockerfile COPY depends on it
   - Do NOT exclude `infra/alembic/` -- Dockerfile COPY depends on it
   - Exclude `tools/` -- contains dev-only Tailwind binary (~60MB)
   - Exclude `.agents/` if it exists -- dev tooling
   - Exclude `*.pyc` and `*.pyo` from rsync (will compile fresh)

   The rsync command:
   ```bash
   rsync -a \
     --exclude='.git/' \
     --exclude='.github/' \
     --exclude='.claude/' \
     --exclude='.planning/' \
     --exclude='.vscode/' \
     --exclude='.playwright-mcp/' \
     --exclude='.pytest_cache/' \
     --exclude='.venv/' \
     --exclude='.agents/' \
     --exclude='tests/' \
     --exclude='plan/' \
     --exclude='gsd-template/' \
     --exclude='node_modules/' \
     --exclude='__pycache__/' \
     --exclude='*.pyc' \
     --exclude='*.pyo' \
     --exclude='dist/' \
     --exclude='backups/' \
     --exclude='tools/' \
     --exclude='.env' \
     --exclude='.env.production' \
     --exclude='.env.production.example' \
     --exclude='docker-compose.prod.yml' \
     --exclude='scripts/generate_license.py' \
     --exclude='CLAUDE.md' \
     --exclude='GUIDE.md' \
     --exclude='pytest.ini' \
     --exclude='.gitignore' \
     ./ "$STAGE/"
   ```

7. **Copy license.key if present:**
   ```bash
   if [ -f "license.key" ]; then
     cp license.key "$STAGE/"
     info "Included license.key in distribution"
   fi
   ```
   (rsync may already handle this if license.key is in root and not excluded, but explicit copy ensures it. If rsync already copies it, the cp is a harmless overwrite.)

8. **Compile licensing module to .pyc (CRITICAL -- legacy placement):**
   ```bash
   python3 -m compileall -b -f -q "$STAGE/core/licensing/"
   ```
   The `-b` flag is MANDATORY. Without it, .pyc goes to `__pycache__/` and will NOT import when .py is removed. See RESEARCH.md Pitfall 1.

9. **Remove .py source from licensing module ONLY:**
   ```bash
   rm "$STAGE/core/licensing/__init__.py"
   rm "$STAGE/core/licensing/license_manager.py"
   rm -rf "$STAGE/core/licensing/__pycache__/"
   ```

10. **Import validation test against STAGED directory (not source tree):**
    ```bash
    PYTHONPATH="$STAGE" python3 -c "import core.licensing; import core.licensing.license_manager"
    ```
    Use this import form -- NOT `from core.licensing import validate_license` which fails because `__init__.py` does not re-export it. See RESEARCH.md Pitfall 3.

11. **Warn about placeholder API key in .env.example:**
    ```bash
    if [ -f "$STAGE/.env.example" ]; then
      if grep -qE 'GOOGLE_MAPS_API_KEY\s*=\s*$' "$STAGE/.env.example" || \
         grep -qE 'GOOGLE_MAPS_API_KEY\s*=\s*your' "$STAGE/.env.example"; then
        warn ".env.example has placeholder GOOGLE_MAPS_API_KEY -- update before customer delivery"
      fi
    fi
    ```

12. **Create tarball:**
    ```bash
    mkdir -p dist
    tar czf "dist/$DIST_NAME-$VERSION.tar.gz" -C "$BUILD_DIR" "$DIST_NAME/"
    ```

13. **Report success with size:**
    ```bash
    SIZE=$(du -h "dist/$DIST_NAME-$VERSION.tar.gz" | cut -f1)
    success "Built dist/$DIST_NAME-$VERSION.tar.gz ($SIZE)"
    ```

14. **Make executable:** `chmod +x scripts/build-dist.sh`

**Update `core/licensing/__init__.py`:** Change line 28 from:
```
      distribution (see Makefile `make dist`). This isn't real protection
```
to:
```
      distribution (see `scripts/build-dist.sh`). This isn't real protection
```
