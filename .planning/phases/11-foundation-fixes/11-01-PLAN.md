<objective>
Improve the clean_cdcms_address() regex pipeline to split trailing uppercase letters from concatenated CDCMS text and reorder abbreviation expansion to run after word splitting.

Purpose: CDCMS addresses like "ANANDAMANDIRAMK" are concatenated without separators. The current pipeline has no regex to detect trailing short letter groups stuck to longer words. Additionally, standalone abbreviation patterns like \bPO\b only work at word boundaries, so word splitting must happen first.

Output: Enhanced clean_cdcms_address() with new regex step and reordered abbreviation expansion, plus comprehensive tests proving correctness and zero regressions.
</objective>
