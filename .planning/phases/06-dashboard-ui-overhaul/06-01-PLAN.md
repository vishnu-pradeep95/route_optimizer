Rewrite App.css to implement the 3-tier responsive sidebar:

1. **Desktop (>= 1280px):** Full 220px sidebar, always visible. Labels and brand text visible.
   - `.app-sidebar` width: `var(--sidebar-expanded)` (220px)
   - Remove position: fixed -- switch to CSS Grid column-based layout
   - Grid: `grid-template-columns: var(--sidebar-expanded) 1fr`
   - `.sidebar-nav-label`, `.sidebar-brand-text`, `.sidebar-health-label` all visible (opacity: 1)

2. **Tablet (< 1280px):** Collapsed to 64px icon-only strip.
   - `.app` grid changes to `grid-template-columns: var(--sidebar-collapsed) 1fr`
   - `.app-sidebar` width: `var(--sidebar-collapsed)` (64px)
   - `.sidebar-nav-label`, `.sidebar-brand-text`, `.sidebar-health-label` hidden (display: none)

3. **Mobile (< 768px):** Sidebar hidden entirely, DaisyUI drawer takes over.
   - `.app` grid changes to `grid-template-columns: 1fr` (no sidebar column)
   - `.app-sidebar` display: none
   - `.mobile-menu-btn` display: flex (hidden at larger breakpoints)

CSS approach -- use mobile-first with min-width breakpoints:

```css
/* Base: mobile -- no sidebar, drawer handles navigation */
.app {
  display: grid;
  grid-template-columns: 1fr;
  height: 100vh;
  overflow: hidden;
}

.app-sidebar {
  display: none; /* Hidden on mobile -- drawer replaces it */
}

.mobile-menu-btn {
  display: flex;
  align-items: center;
  justify-content: center;
  position: fixed;
  top: 12px;
  left: 12px;
  z-index: 50;
  width: 40px;
  height: 40px;
  border-radius: var(--radius-md);
  background: var(--color-surface-dark);
  color: var(--color-text-inverse);
  border: none;
  cursor: pointer;
}

/* Tablet: icon-only sidebar */
@media (min-width: 768px) {
  .app {
    grid-template-columns: var(--sidebar-collapsed) 1fr;
  }
  .app-sidebar {
    display: flex;
    flex-direction: column;
    width: var(--sidebar-collapsed);
    /* Keep existing background, color, overflow styles */
  }
  .mobile-menu-btn {
    display: none;
  }
  .sidebar-nav-label,
  .sidebar-brand-text,
  .sidebar-health-label {
    display: none;
  }
}

/* Desktop: full sidebar */
@media (min-width: 1280px) {
  .app {
    grid-template-columns: var(--sidebar-expanded) 1fr;
  }
  .app-sidebar {
    width: var(--sidebar-expanded);
  }
  .sidebar-nav-label,
  .sidebar-brand-text,
  .sidebar-health-label {
    display: inline; /* or block, depending on context */
    opacity: 1;
  }
}
```

Key changes from current CSS:
- REMOVE `.app-sidebar.expanded` class (no JS hover state)
- REMOVE `position: fixed` on sidebar (use grid-based layout instead -- sidebar is IN the grid flow)
- REMOVE all opacity transitions tied to `.expanded` class
- REMOVE `grid-column: 2` on `.app-main` (grid auto-places it)
- KEEP existing visual styles (background, colors, nav item styles, health dot, border treatments)
- KEEP `.sidebar-nav-icon` width: 32px and text-align: center for alignment

The sidebar nav icon container (`.sidebar-nav-icon`) needs to accommodate SVG components instead of emoji. Remove `font-size: 18px` (SVG uses width/height), keep `width: 32px`, add `display: flex; align-items: center; justify-content: center;`.

Similarly `.sidebar-brand-icon` -- remove `font-size: 22px`, add flex centering for the Fuel SVG.
