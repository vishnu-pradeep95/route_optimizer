<objective>
Enrich the upload_and_optimize endpoint to collect geocoding failures and return structured import diagnostics.

Purpose: The endpoint currently logs geocoding failures and raises HTTPException(400) when zero orders geocode. This plan threads errors from CsvImporter's new ImportResult through the geocoding loop, collects geocoding-stage failures with user-friendly reasons, and returns an enriched OptimizationSummary with per-row failure details and summary counts. The zero-success case returns a structured 200 response instead of a 400 error.

Output: Modified main.py with ImportFailure model, enriched OptimizationSummary, geocoding error collection, and zero-success handling.
</objective>
