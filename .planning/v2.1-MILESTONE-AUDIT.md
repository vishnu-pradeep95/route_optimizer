## Remaining Minor Issues

### 1. X-License-Expires-In missing from CORS expose_headers

**Severity:** LOW — affects cross-origin browser clients only
**Affects:** LIC-02

`enforcement.py` sets `X-License-Expires-In` on all response paths, but `main.py:322` `expose_headers` only lists `["X-License-Warning", "Retry-After", "X-Request-ID"]`. Cross-origin browser clients (dashboard on different port/domain) cannot read this header. Same-origin and server-side monitoring consumers are unaffected.

Also missing: `X-License-Status` (set on /health path only).

### 2. E2E Integrity Tamper Test Skipped in CI

**Severity:** INFORMATIONAL — by design
**Affects:** DOC-01

`security-pipeline.spec.ts` Scenario 3 uses `test.skip(!isProductionBuild)` and will always skip in CI (dev builds have empty `_INTEGRITY_MANIFEST`). The test is correctly authored and requires a `build-dist.sh` production artifact to exercise.
